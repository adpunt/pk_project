## About The Project

We can use pharmacokinetic models to analyse how a drug passes through a patient's body. It looks at the drug from delivery to diffusion to clearance. The patient's body is modelled by compartments through which the drug in question passes. Such models can be used to both determine dosing and delivery protocols, and to educate people on the process of drug delivery.
